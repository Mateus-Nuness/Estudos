# 📚 Repositório de Estudos & Projetos

Este repositório centraliza minha jornada de aprendizado em desenvolvimento de software, dividida entre estudos teóricos, exercícios práticos e projetos autorais. 

Atualmente, meu foco principal está no desenvolvimento de soluções robustas para o **Backend utilizando Java**, além de consolidar conceitos essenciais de **Frontend**.

---

## 📂 Estrutura do Repositório

### 🖥️ [Back-end](./Back-and)
Pasta dedicada aos conceitos de desenvolvimento do lado do servidor:
*   **Linguagem principal:** Java.
*   *Estudos futuros:* Arquitetura de APIs, frameworks (Spring Boot) e integração com bancos de dados relacionais (SQL).

### 🎨 [Front-end](./Front-end)
Exercícios práticos de interface e lógica client-side:
*   Estruturação de layouts, estilização e manipulação básica do DOM com JavaScript.

### 🚀 Projetos em Destaque

#### 📱 [Projeto 1 Pokedex](./Projeto%201%20Pokedex)
*   **Descrição:** Aplicação web interativa que contém dados sobre pokémons, focada em manipulação de APIs e exibição dinâmica de dados no frontend.
*   **Tecnologias:** HTML, CSS e JavaScript.

#### 🃏 [Pegadinha Botão Andante](./Pegadinha_Botão_Andante)
*   **Descrição:** Um projeto divertido de lógica de programação onde um botão "foge" do cursor do usuário quando ele tenta clicar, aplicando conceitos de eventos dinâmicos em tela.
*   **Tecnologias:** HTML, CSS e JavaScript.
